# DataAsset Sheet Editor

![UE Version](https://img.shields.io/badge/Unreal%20Engine-5.7-0e1128?logo=unrealengine&logoColor=white)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

---

DataAsset Sheet Editorは、Unreal Engine 5向けのエディタ拡張プラグインです。
複数のDataAssetをスプレッドシート形式で一覧表示し、効率的に閲覧・編集することができます。

## 特徴

- 同一クラスの複数DataAssetをテーブル形式で一覧表示
- セルをクリックして直接編集
- プロパティの型に応じた適切なエディタウィジェット
- Undo/Redo対応
- フィルタリング・ソート機能

## 動作環境

- Unreal Engine 5.7
- Windows (Win64)

## インストール

1. [Releases](https://github.com/pafuhana1213/DataAssetSheetEditor/releases)から最新版をダウンロード
2. プロジェクトの`Plugins/`フォルダに`DataAssetSheetEditor/`を配置
3. UEエディタを再起動

## 使い方

1. UEエディタのメニューから `Window > DataAsset Sheet Editor` を選択
2. 編集したいDataAssetのクラスを選択
3. スプレッドシート上で値を直接編集
